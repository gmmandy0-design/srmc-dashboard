# 🏥 SRMC Dashboard

Streamlit-based growth and operations dashboard for **Schotters Road Medical Centre (SRMC)**.

---

## 🚀 Overview
This platform provides real-time insights into:
- Patient acquisition
- Booking performance
- Operational workload
- Clinic growth strategy

It integrates **analytics, booking systems, and decision support** into one unified dashboard.

---

## ✅ Features

### 📊 Dashboard
- KPI tracking (patients, revenue, channels)
- Growth trends and charts
- Competitor benchmarking

### 📅 Booking System
- Patient appointment request form
- Secure data storage (SQLite)
- Role-based access (Owner / Manager / Staff)

### 📲 Patient Access
- QR code booking
- Direct HotDoc integration

### 📈 Analytics
- Booking conversion tracking
- Patient acquisition funnel
- Workload monitoring

### 📄 Reports
- PDF export (Owner role only)
